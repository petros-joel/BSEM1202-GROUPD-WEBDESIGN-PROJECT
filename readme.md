GROUP D HEALTH SERVICES
Sierra Leone's Premier Healthcare Platform
===========================================

PROJECT OVERVIEW
----------------
Group D Health Services is a comprehensive web-based healthcare management system designed to address healthcare challenges in Sierra Leone. The platform provides intelligent doctor matching, online appointment booking, patient portal, online pharmacy, and emergency services across 12 locations nationwide.

SMART FEATURES
--------------
Specialization Matching: System recommends correct doctor based on symptoms
Date and Time Conflict: Prevents double-booking same doctor
Time Slots: Morning (8AM to 12PM), Afternoon (1PM to 5PM), Night (6PM to 10PM)
Medical vs Finance Separation: Finance manager handles all financial inquiries
AI Chat Assistant: Answers health-related questions automatically
LocalStorage Persistence: All data saved in browser

INSTALLATION
------------
1. Clone the repository:
   git clone https://github.com/YOUR-USERNAME/groupd-health-services.git

2. Navigate to project folder:
   cd groupd-health-services

3. Open index.html in your browser

DEPLOYMENT ON GITHUB PAGES
--------------------------
1. Push code to GitHub repository
2. Go to Settings > Pages
3. Select main branch as source
4. Click Save
5. Wait 1-2 minutes for deployment

Live URL: https://YOUR-USERNAME.github.io/groupd-health-services/
